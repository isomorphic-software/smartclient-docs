## Attr: StyleSetting.editorProperties

### Description
FormItem properties to apply to editors for this setting.

**Flags**: IR

---
## Attr: StyleSetting.settingName

### Description
The CSS attribute name used when generating CSS from this item. Rarely used to override the default behavior of using the [name](#attr-stylesettingname).

**Flags**: IR

---
## Attr: StyleSetting.defaultEditorType

### Description
The type of editor to use for this setting.

**Flags**: IR

---
## Attr: StyleSetting.name

### Description
Unique name that identifies this setting.

**Flags**: IR

---
## Attr: StyleSetting.title

### Description
Title for this setting when applied to a FormItem.

**Flags**: IR

---
## Attr: StyleSetting.groupName

### Description
Name of the [group](../main.md#object-stylegroup) this setting will appear in for editing.

**Flags**: IR

---
## Attr: StyleSetting.showAsymmetry

### Description
If this setting allows [asymmetric editing](#attr-stylesettingallowasymmetry), should it be edited in that mode by default?

**Flags**: IR

---
## Attr: StyleSetting.allowAsymmetry

### Description
Can this setting be edited asymmetrically? For example, a border-setting can be applied to all sides at once, or asymmetrically to each side, as four separate border-settings.

**Flags**: IR
