## KB Topic: gridHeader

### Description
Properties and methods related to the ListGrid header. ListGrid headers are implemented as a [Toolbar](../classes/Toolbar.md#class-toolbar) of buttons shown at the top of the ListGrid (one button per column).  
The toolbar header provides UI for interacting with the ListGrid fields directly (sorting, resizing, reordering columns, etc).
