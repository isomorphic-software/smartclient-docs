## KB Topic: submitting

### Description
Direct submission of forms to a target URL

**NOTE:** directly submitting forms is only done for specialized purposes, such as integration with certain legacy systems. Normal form usage contacts the server via [DataBound Component Methods](dataBoundComponentMethods.md#kb-topic-databound-component-methods), through the RPCManager system.
